1. One technical concept that I developed a greater mastery of was the coding language **HTML**. When designing this **portfolio**, I had to split the webpage into **sections**. I was able to achieve this by assigning elements with **attributes** like `Id` and `class` to separate elements with unique properties

```html
   <header id="here">
    <a id="top" href="#section-a">JCA.DEV</a> // Id is only assigned to this, allowing us to only modify this property

    <nav id="stick">
      <a class="goto" href="#home">HOME</a>
      <a class="goto" href="#about">ABOUT</a>
      <a class="goto" href="#projects">PROJECTS</a>
      <a class="goto" href="#contact">CONTACT</a>
    </nav>
  </header>
```
